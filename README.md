# CODSOFT Machine Learning Projects

4 Machine Learning projects with fully interactive, custom dark-themed desktop GUIs.

## 📁 Repository Structure

- **Task1_Movie_Genre/**: Movie Genre Classification using TF-IDF and Naive Bayes.
- **Task2_Credit_Card_Fraud/**: Credit Card Fraud Detection Engine.
- **Task3_Customer_Churn_Prediction/**: Customer Churn Prediction Engine.
- **Task4_Spam_SMS_Detection/**: AI Spam SMS Detection Engine.
